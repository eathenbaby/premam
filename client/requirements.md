## Packages
framer-motion | Essential for the paper fold, tab switching, and bouquet reveal animations
lucide-react | For elegant icons (Feather, Heart, etc.)
canvas-confetti | For the "Sent with love" celebration effect
clsx | Utility for conditional class names (standard in shadcn)
tailwind-merge | Utility for merging tailwind classes (standard in shadcn)
date-fns | For formatting timestamps nicely

## Notes
- Fonts: Cormorant Garamond (Headlines), Lora (Body), DM Sans (UI) will be loaded via Google Fonts in index.css
- Images: Using Unsplash for bouquet placeholders and textures if needed.
- Theme: "Botanical Letter Press" - heavily reliant on CSS variables for paper textures and ink colors.
